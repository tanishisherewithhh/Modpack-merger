# Modpack-merger
Merge modrinth modpacks directly in a smart way to avoid conflicts.

## How to Use
1.  Go to the [Live Tool](https://tanishisherewithhh.github.io/Modpack-merger/).
2.  Drag and drop your `.mrpack` files into the upload zone.
3.  Arrange the packs in order of importance using the ▲ and ▼ buttons.
4.  Ensure there are no red "Logic Mismatch" errors.
5.  Click **Merge & Export** to download your new pack.
