    Frequently Asked Questions

    What is a modpack merger or pack combiner?
    A modpack merger (also called a pack combiner) is a tool that lets you combine multiple Minecraft modpacks into a single pack. This tool supports Modrinth .mrpack files and standard .zip files, automatically resolving mod conflicts and duplicates based on a priority system you control.
    How do I merge modpacks on Modrinth?
    To merge Modrinth modpacks, download the .mrpack files you want to combine, then upload them to this tool. Arrange the packs in priority order (the top pack takes precedence), review the merged mod list, and export as a new .mrpack or .zip file. The tool handles duplicate detection and conflict resolution automatically.
    How do I combine modpacks in Modrinth?
    You can combine Modrinth modpacks using this online ModPack Merger tool. Simply drag and drop your .mrpack files, set which pack has higher priority for overlapping mods, optionally run Deep Analysis to detect version conflicts, then export your combined modpack in either .mrpack or .zip format.
    Can I use this as a Minecraft mod combiner?
    Yes! This tool works as a complete Minecraft mod combiner. It supports Fabric, Forge, NeoForge, Quilt, and LiteLoader modpacks. You can merge any combination of .mrpack and .zip mod packs, and the tool will intelligently handle duplicate mods, version conflicts, and dependency issues.
    Does this work as a resource pack combiner?
    While this tool is primarily designed for modpacks, it also handles resource packs, shader packs, and config files bundled within your modpacks. Any resource packs included in the overrides folder of your .mrpack or .zip files will be merged along with the mods.
    How do I convert a Modrinth pack to ZIP?
    To convert a Modrinth .mrpack file to a standard .zip, upload your .mrpack file to this tool and click "Export ZIP" instead of "Export MRPACK". The tool will download all the mods and package everything into a standard .zip file that can be used with any Minecraft launcher.
    Is this tool free to use?
    Yes, this ModPack Merger is completely free and open-source. It runs entirely in your browser, no files are uploaded to any server. All processing happens locally on your device, keeping your modpacks private and secure.
    What file formats are supported?
    This tool supports Modrinth .mrpack files and standard .zip files. It can export merged packs in both formats. The .mrpack format is compatible with Modrinth-based launchers, while the .zip format works with most other Minecraft launchers.

    How to Merge Minecraft Modpacks

    1. Upload your modpacks — Click the drop zone or drag and drop your .mrpack or .zip files. You can add as many packs as you need.
    2. Arrange pack priority — Use the UP/DOWN buttons to reorder packs. The top pack has the highest priority and its mods will take precedence over lower packs.
    3. Review merged contents — Browse through the Mods, Resource Packs, Shader Packs, Configs, and Others tabs to see all files from your combined packs.
    4. Resolve conflicts — Duplicates are automatically resolved by priority. Optionally click "Deep Analyze JARs" for advanced version conflict detection and dependency validation.
    5. Export your merged pack — Click "Export MRPACK" for Modrinth-compatible format or "Export ZIP" for a standard zip file. Your merged modpack downloads automatically.
